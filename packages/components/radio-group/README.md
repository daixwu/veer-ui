# RadioGroup 组件

按钮用于开始一个即时操作。

## 基本用法

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

## Props 配置

| 属性 | 说明 | 类型 | 默认值 | 可选值 |
| - | - | - | - | - |
| disabled | 按钮失效状态 | Boolean | false | true/false |

## 插槽

| 名字 | 说明 |
| - | - |
| prepend | 前置内容 |
| append | 后置内容 |

## 事件

| 事件名 | 说明 | 参数 |
| - | - | - |
| focus | 输入框聚焦后触发此事件，如果禁用状态，则不触发 | e - 事件对象
