# Switch 滑动开关

滑动开关，用于切换 on/off 状态。

# 示例

  • 基本用法

    <cube-switch v-model="value">
  Switch
</cube-switch>

    export default {
  data() {
    return {
      value: true
    }
  }
}

    value值true对应着 on 状态（开） ，value值false对应着 off 状态（关）。

  • 禁用状态

    <cube-switch v-model="value" :disabled="true">
  Disabled Switch
</cube-switch>

# Props 配置

参数 说明 类型 可选值 默认值
v-model 开关状态，双向数据绑定 Boolean true/false false
disabled 是否禁用 Boolean true/false false

# 事件

事件名 说明 参数
input 绑定值变化时触发 更新后的绑定值
